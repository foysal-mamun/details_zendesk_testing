On Distribution Account / CURRENT STATEMENTS you can add your sales import setups.If a distribution account sends you sales statements and cost statements, you need to add an import setup for these in order to import them on DISTRIBUTION / IMPORTS.All import setups (Digital, Physical, Costs, etc) are listed here.To add a new import setup to your account, click the green PLUS icon on top right of the Statement setup box. Now enter a description (important when the distribution account has several statement types such as streaming and download, different territories etc, define here).Select the invoice project, type digital or physical, currency and period for which you receive statements. These are the mandatory infos we need to set up the sales import.Enter start and end dates when you import the first and when to expect the last (this can always be amended later). For the set period and start and end time you will now find the yellow boxes (import to do) on DISTRIBUTION / IMPORTS.Select a DSS list from the drop down menu if applicable. For some main distribution accounts ''details'' already supplies presets for the statement import.Add the distribution fee on sales and returns if applicable. Please note distribution fees are only applied on PPDs - not on PPUs.::::Click NEXT and choose the invoice contact for statement invoices. Click OK to confirm.You can edit the setup settings by clicking on the statement name at any time.To delete a statement setup, click the red X icon.
