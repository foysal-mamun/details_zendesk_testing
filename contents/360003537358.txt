On DISTRIBUTION / PLUGINS multiple plugins can be activated to make your daily distribution work even easier.At the moment these plugins are availableCheck Statement Matching: Export Imported lines by Identifier with matching InfosCustomized Release Report Plugin:This plugin enables you to generate customized release reports according to a bespoke template created for the purpose.Order Discount:This plugin enables to add discounts in distribution orders.Order Import (XLS):This plugin enables importing xls file for distribution orders.Automatic Statement Importing:This plugin provides automatic statement importing for multiple shop/distributor statement formats. To activate a plugin just click the green ACTIVATE button right next to the plugin name.More plugins will be added in the future.
